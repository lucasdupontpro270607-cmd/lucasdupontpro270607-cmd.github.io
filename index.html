Ajout prévu : Synchroniser un compte

But : permettre à l’utilisateur de récupérer son profil sur téléphone ou PC avec un code de sauvegarde, sans serveur, en mode hors ligne.

Fonctionnement

  • Depuis un appareil déjà connecté, l’utilisateur clique sur Générer un code de synchronisation.
  • L’application encode toutes les données du compte (profil, séances, amis, boutique, quêtes, duels, paramètres).
  • Elle génère un code long copiable.
  • Sur l’autre appareil, dans l’écran de connexion, l’utilisateur clique sur Synchroniser un compte.
  • Il colle le code, choisit un nom de compte local et importe les données.
  • Le profil est alors recréé dans le localStorage de l’appareil, donc utilisable hors ligne.

Important

Ce n’est pas une vraie synchro temps réel : c’est un transfert manuel de sauvegarde entre appareils, sans serveur.

À coder dans la vraie version

  • Un bouton Synchroniser un compte dans la zone connexion.
  • Une zone textarea pour coller le code.
  • Une fonction generateSyncCode().
  • Une fonction importSyncCode().
  • Une option pour écraser ou créer un compte local.
